# LabeledContent

`<LabeledContent />` is a component that displays a label and a content area. It is used to display a label and a content area.
The reason to use this component is to make the content more readable and to make the content more accessible.

- unified styling: the component has a unified styling that is used throughout the app
- accessible: `aria-labelledby` and `role="region"` are used to make the component more accessible
